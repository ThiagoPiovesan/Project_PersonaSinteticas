# 🌌 Galaxies — Resultados Parte 1 - ML

## Quantidade de Clusters encontrados

- Foram identificados 2 clusters através do algoritmo K-Means, validados também com GMM. Apesar dos índices de silhouette indicarem baixa separação natural nos dados (0.096 e 0.030 respectivamente), a análise estatística revelou diferença significativa na frequência de compra entre os grupos (χ²=3000, p≈0), que se tornou o principal eixo de diferenciação.

## Perfil principal de cada cluster

- Cluster 0 — "Comprador Ativo" (1.273 consumidores, 42.4%)
Consumidor predominantemente masculino, com idade média de 44 anos, ticket médio de R$122 e média de 4 itens por compra. Seu grande diferencial é a frequência semanal de compras (56.5%), sugerindo um perfil de reposição contínua. Prefere loja física, categoria de bebidas e pagamento via crédito. Concentrado na região Sudeste.

- Cluster 1 — "Comprador Esporádico" (1.727 consumidores, 57.6%)
Perfil demograficamente similar ao Cluster 0 — masculino, ~45 anos, ticket de R$121 — mas com comportamento de compra trimestral (34.5%), indicando compras planejadas e menos frequentes. Também prefere loja física, bebidas e crédito, com maior concentração no Sudeste.

## Potencias aplicações para a Galaxies

- Retenção e engajamento: acionar o Cluster 1 com campanhas de reativação e incentivos de frequência, buscando migrar esses consumidores para um comportamento mais próximo ao do Cluster 0

- Fidelização: para o Cluster 0, programas de fidelidade e benefícios por volume fazem sentido, dado o padrão de compra recorrente

- Personalização de comunicação: o agente de IA pode adotar tom e urgência distintos para cada perfil — conversacional e de reposição para o Ativo, consultivo e de impacto para o Esporádico

- Planejamento de estoque e oferta: o Cluster 0 sustenta demanda contínua enquanto o Cluster 1 pode gerar picos sazonais, permitindo estratégias de abastecimento diferenciadas
